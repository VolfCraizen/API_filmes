Documentation de API Films

Cet API permet d'afficher selon les désires de l'utilisateur, ajouter, modifier et supprimer des filmes. Elle permet aussi la création d'utilisateur et la connexion d'utilisateur.

Les routes sont accessible à partir de cette url : https://projet-films.onrender.com/



Routes


Récupérer la liste de films

GET api/films/

https://projet-films.onrender.com/api/films

Paramètres

https://projet-films.onrender.com/api/films?subject=titre&order-direction=desc&limit=2

Subject

  • titre
  • annee
  • realisation

order-direction

  • asc
  • desc

limit

  • Nombre entier


Récupérer un film spécifique

GET api/films/:id

https://projet-films.onrender.com/api/films/:id

Nécéssite le paramètre :id (l'identifiant du film)



Ajout d'un film

POST api/films

https://projet-films.onrender.com/api/films

Paramètres

{

titre: "Titre",

genres: ["genre1", "genre2],

description: "Description",

annee: "Annee",

realisation: "Nom du réalisateur",

titreVignette: "Titre de la vignette"

}

Consignes des paramètres

  • Chaque paramètre doit être remplit
  • annee doit être un INT et doit être 4 caractères


Modification d'un film

PUT api/films/:id

https://projet-films.onrender.com/api/films/:id

Nécéssite le paramètre :id (l'identifiant du film)

Paramètres

{

titre: "Titre",

genres: ["genre1", "genre2],

description: "Description",

annee: "Annee",

realisation: "Nom du réalisateur",

titreVignette: "Titre de la vignette"

}

Consignes des paramètres

  • Chaque paramètre doit être remplit
  • annee doit être un INT et doit être 4 caractères


Suppression d'un film

DEL api/films/:id

https://projet-films.onrender.com/api/films/:id

Nécéssite le paramètre :id (l'identifiant du film)



Ajout d'un utilisateur

POST api/utilisateurs/inscription

https://projet-films.onrender.com/api/utilisateurs/inscription

Paramètres

{

courriel: "Adresse courriel",

mdp: "Mot de passe"

}

Consignes des paramètres

  • Chaque paramètre doit être remplit
  • courriel doit être un courriel Ex: exemple@gmail.com
  • mpd doit contenir minimum 8 caractères
  • mpd doit contenir maximum 20 caractères
  • mpd doit contenir minimum 1 majuscule
  • mpd doit contenir minimum 1 minuscule
  • mpd doit contenir minimum 1 chiffre
  • mpd doit contenir minimum 1 symbol
